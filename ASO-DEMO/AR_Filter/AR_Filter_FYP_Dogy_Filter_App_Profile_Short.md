# App Profile - AR Filter: FYP Dogy Filter

## App Profile

- **App name:** AR Filter: FYP Dogy Filter

- **Core feature:** AR dog / dogy / 3D AR character filter app for creating funny short videos. Users can place animated AR characters in the real environment, control character actions, dance with AR characters, and try weird face effects.

- **Main user intent:** Users want to create playful, meme-style short videos using an AR Dogy / AR dog / virtual 3D character filter, especially for funny, trend-inspired, or FYP-style video content.

- **Strong supporting intents:**  
  AR dog filter, dogy filter, virtual dog camera, 3D AR character filter, interactive AR character, funny AR video, meme dog video, FYP-style dog filter, trending AR effect, funny face effect, weird face filter, dance with AR character, short video effect.

- **Not relevant intents:**  
  Generic AR filter without dog/character intent, beauty filter, makeup filter, selfie editor, normal photo editor, pet care app, real dog training app, dog game, dog wallpaper, generic camera app, generic video editor, face filter only, anime/cartoon filter unrelated to AR dogy, platform-specific TikTok/Instagram/Snapchat affiliation intent.
